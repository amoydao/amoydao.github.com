用hexo搭建blog

发表于 2015-10-01   |   分类于 器   |  

环境准备

安装 git
安装 node.js
安装 sublime text
安装 hexo：（Git Bash） $ npm install -g hexo-cli

#博客安装
1.新建博客文件夹（可建在U盘里）
2.在该文件夹上右键运行Git Bash，安装hexo

$ npm install hexo –save
$ hexo init
$ npm install

3.查看结果

$ hexo s

安装主题

主题推荐单：有那些好看的hexo主题?
使用主题：iissnan/hexo-theme-next
克隆安装：

$ git clone https://github.com/iissnan/hexo-theme-next themes/next

启用主题：打开 站点配置文件，找到 theme 字段，并将其值更改为 next
验证主题：

$ hexo s –debug

并访问 http://localhost:4000 ，确保站点正确运行

#设置主题/站点/插件
5分钟快速安装

发布到Github

部署前需要配置_config.yml文件。
deploy:
type: github
repository: https://github.com/amoydao/amoydao.github.io.git #在第二步中你创建的GitHub仓库(repository)路径
brach: master
之后执行下列指令即可完成部署，注意部署会覆盖掉你之前在版本库中存放的文件。

$ hexo clean # 清空
$ hexo g # 本地部署，生成静态页面至hexo\public\目录
$ hexo s # 启动本地服务，进行文章预览调试，http://localhost:4000
$ hexo d # 发布到GitHub上，需要根据提示输入账号和密码

注意：命令必须在init目录下执行，否则不成功，但是也不报错。

若遇到问题：
hexo ERROR: Deployer not found: github
1.安装 npm install hexo-deployer-git –save
2.将deploy 的 type由github改为git
